# Meu Primeiro Site

Este é um projeto simples de um site pessoal desenvolvido em HTML e CSS. O objetivo é apresentar informações básicas sobre mim, Tiago Ferreira, incluindo minha profissão, hobbies e valores pessoais.

## Funcionalidades

- Página inicial com apresentação pessoal
- Lista de interesses e hobbies
- Design responsivo e moderno
- Botão de chamada para ação

## Tecnologias Utilizadas

- **HTML5**: Estrutura da página
- **CSS3**: Estilização e layout responsivo

## Como Executar

1. Clone ou baixe os arquivos do projeto
2. Abra o arquivo `index.html` em qualquer navegador web
3. O site será carregado automaticamente
